Clifford Function Reference

horzcat

Horizontal concatenation.
(Clifford overloading of standard MATLAB® function)

Description

horzcat(x,y) horizontally concatenates two elements, x and y into an array. Normally this function is not used directly (it implements concatenation with [ and ]).

Examples

>> clifford_signature(2,2)
>> show(horzcat(e1,e2))
 
e1 *
 
     1     0

+ e2 *
 
     0     1

>> show([e1,e2])
 
e1 *
 
     1     0

+ e2 *
 
     0     1
