Clifford Function Reference

clifford_conjugate

Computes the Clifford conjugate of a multivector.

Description

clifford_conjugate(x) computes the Clifford conjugate of a multivector which reverses the sign of the blades of grades 1 mod 4 and 2 mod 4, while maintaining the signs of all othe blade components.

Examples

>> clifford_signature(2,1)
>> x = randm
 
x =
 
 - 0.0852 e0 
 - 0.2013 e1  + 0.2661 e2  + 0.2608 e3 
 - 0.7211 e12 - 0.0251 e13 - 0.1375 e23
 + 0.5233 e123
 
>> clifford_conjugate(x)
 
ans =
 
 - 0.0852 e0 
 + 0.2013 e1  - 0.2661 e2  - 0.2608 e3 
 + 0.7211 e12 + 0.0251 e13 + 0.1375 e23
 + 0.5233 e123
