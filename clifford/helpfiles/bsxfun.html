Clifford Function Reference

bsxfun

Binary Singleton Expansion Function.
(Clifford overloading of standard MATLAB® function)

Description

bsxfun(fun,x,y) Applies an element by element binary operation, denoted by fun, on two Clifford objects, x and y.

Examples

>> clifford_signature(1,1)
x =
 
   0.1389 e0
 - 0.3224 e1 + 0.5852 e2
 - 0.7309 e12
 
>> y = randm
 
y =
 
 - 0.1973 e0
 - 0.4659 e1 + 0.6160 e2
 + 0.6037 e12
 
>> bsxfun(@plus,x,y)
 
ans =
 
 - 0.0584 e0
 - 0.7884 e1 + 1.2012 e2
 - 0.1272 e12
