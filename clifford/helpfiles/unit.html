Clifford Function Reference

unit

multivector. Divides a multivector by its own modulus. The result is a multivector with unit modulus.

Description

unit(x) divides multivector x by its own modulus and returns the multivector answer.

Examples

>> clifford_signature(1,1)
>> x = clifford(1,2,3,4)
 
x =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 + 4.0000 e12
 
>> unit(x)
 
ans =
 
   0.1826 e0
 + 0.3651 e1 + 0.5477 e2
 + 0.7303 e12
