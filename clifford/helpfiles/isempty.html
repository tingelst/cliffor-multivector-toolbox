Clifford Function Reference

isempty

True for empty matrix.
(Clifford overloading of standard MATLAB® function)

Description

isempty(x) returns logical true or false depending on whether the argument x is empty.

Examples

Non-empty array: 
>> x = randm(2)
 
x =
 
     2x2 Cl(1,1) multivector array
 
>> isempty(x)

ans =

     0
Empty array: 
>> x = clifford.empty
 
x =
 
     [] Cl(1,1) multivector
 
>> isempty(x)

ans =

     1
