Clifford Function Reference

rdivide

./ Right array divide.
(Clifford overloading of standard MATLAB® function)

Description

rdivide(x,y) divides element x by element y.

Examples

>> clifford_signature(1,1)
>> x = clifford(1,2,3,4)
 
x =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 + 4.0000 e12
 
>> y = clifford(5,6,7,8)
 
y =
 
   5.0000 e0
 + 6.0000 e1 + 7.0000 e2
 + 8.0000 e12
 
>> rdivide(x,y)
 
ans =
 
   0.6923 e0
 - 0.3077 e1 - 0.6154 e2
 - 0.6154 e12
 
>> rdivide(y,x)
 
ans =
 
   1.8000 e0
 + 0.8000 e1 + 1.6000 e2
 + 1.6000 e12
 
>> a = [x,y]
 
a =
 
     1x2 Cl(1,1) multivector array
 
>> show(a)
 
a =
 
e0 *
 
     1     5

+ e1 *
 
     2     6

+ e2 *
 
     3     7

+ e12 *
 
     4     8

>> rdivide(a,x)
 
ans =
 
     1x2 Cl(1,1) multivector array
 
>> show(ans)
 
ans =
 
e0 *
 
    1.0000    1.8000

+ e1 *
 
    0.0000    0.8000

+ e2 *
 
         0    1.6000

+ e12 *
 
   -0.0000    1.6000
