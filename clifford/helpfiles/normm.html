Clifford Function Reference

normm

Norm of a multivector (norm of each element for arrays)

Description

norm(x) computes the norm of a multivector or multivector array. "We get the norm |M| of a multivector as |M|2 = M * M̅ = ‹MM̅›0 = m2 0 + m2 1 + m2 2 + m2 12."[1]

Examples

>> clifford_signature(1,1)
>> x = 1*e0 - 2*e1 + 3*e2 - 4*e12
 
x =
 
   1.0000 e0
 - 2.0000 e1 + 3.0000 e2
 - 4.0000 e12
 
>> normm(x)

ans =

    30
