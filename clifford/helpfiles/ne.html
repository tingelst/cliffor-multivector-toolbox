Clifford Function Reference

ne

~= Not equal.
(Clifford overloading of standard MATLAB® function)

Description

ne(x,y) computes whether the arguments x and y are unequal and returns true if they are.

Examples

>> x = randm
 
x =
 
   0.7548 e0
 + 0.1521 e1 + 0.6229 e2
 + 0.1381 e12
 
>> y = randm
 
y =
 
 - 0.6291 e0
 + 0.5436 e1 + 0.0549 e2
 - 0.5529 e12
 
>> ne(x,x)

ans =

     0

>> ne(x,y)

ans =

     1
