Clifford Function Reference

inv

Inverse of a multivector (matrix).
(Clifford overloading of standard MATLAB® function)

Description

inv(x) computes the inverse of any invertable argument multivector x, including that of matricies of multivectors.

Examples

>> clifford_signature(1,1)
>> x = randm(2)
 
x =
 
     2x2 Cl(1,1) multivector array
 
>> show(x*inv(x))
 
e0 *
 
    1.0000   -0.0000
   -0.0000    1.0000

+ e1 *
 
   1.0e-14 *

   -0.0222   -0.0222
    0.6078   -0.0250

+ e2 *
 
   1.0e-14 *

   -0.0333    0.0833
    0.2220    0.0777

+ e12 *
 
   1.0e-14 *

   -0.0111    0.0722
    0.3608    0.0222
If x is not invertable (i.e. is a divisor of zero) the result will contain NaNs or infinities. 
>> clifford_signature(1,1)
>> x = e1 - e2
 
x =
 
   1.0000 e1 - 1.0000 e2
 
>> inv(x)
 
ans =
 
   NaN e0
 - Inf e1 + Inf e2
 + NaN e12
