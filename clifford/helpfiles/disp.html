Clifford Function Reference

disp

Display array.
(Clifford overloading of standard MATLAB® function)

Description

disp(x) displays the value of the argument, x.

Examples

For example if you had previously created values x and y: 
>> disp(x)
 - 0.9401 e0
 - 0.2775 e1 - 0.0973 e2
 + 0.1723 e12
>> disp(y)
 - 0.1855 e0
 + 0.3149 e1 + 0.2783 e2
 - 0.8882 e12
>> clifford_signature(1,1)
>> x = randm(2)
 
x =
 
     2x2 Cl(1,1) multivector array
 
>> disp(x)
     2x2 Cl(1,1) multivector array
