Clifford Function Reference

char

Create character array (string).
(Clifford overloading of standard MATLAB® function)

Description

char(x) creates a text character string representing the input Clifford object, x.

Examples

>> clifford_signature(1,1)
>> x = randm
 
x =
 
   0.6133 e0
 + 0.6223 e1 - 0.4845 e2
 + 0.0434 e12
 
>> class(x)

ans =

clifford

>> x = char(x)

x =

   0.6133 e0 + 0.6223 e1 - 0.4845 e2 + 0.0434 e12

>> class(x)

ans =

char
