Clifford Function Reference

scalar_product

Multivector scalar product. The two arguments must be the same size or one must be scalar. This is an elementwise operation if applied to arrays.

Description

scalar_product(x,y) computes the scalar product of two elements x and y. This is a generalisation of the inner product of vectors. "The symmetric scalar part of the geometric product of two multivectors M, N ∈ Cl(2, 0) is also called their scalar product and because of its fundamental importance denoted with a special product sign ∗ M ∗ N = ‹MN›0 = m0n0 + m1n1 + m2n2 − m12n12 = N ∗ M"[1]

Examples

>> clifford_signature(1,1)
>> x = clifford(1,2,3,4)
 
x =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 + 4.0000 e12
 
>> y = randm
 
y =
 
 - 0.9401 e0
 - 0.2775 e1 - 0.0973 e2
 + 0.1723 e12
 
>> scalar_product(x,x)

ans =

   -20

>> scalar_product(x,y)

ans =

   -1.8924
