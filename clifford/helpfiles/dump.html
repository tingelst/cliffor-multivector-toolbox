Clifford Function Reference

dump

Outputs a human readable dump of the contents of the multivector (for debugging purposes).

Description

dump(x) outputs the current signature of the argument, x, and also the multivector components, which can be used for debugging. The sizes of the components are shown.

Examples

>> x = randm
 
x =
 
   0.0038 e0
 - 0.9163 e1 - 0.1382 e2
 + 0.3758 e12
 
>> dump(x)
Signature: 1  1  0
 
Multivector components:
1: [1  1]  double
2: [1  1]  double
3: [1  1]  double
4: [1  1]  double
