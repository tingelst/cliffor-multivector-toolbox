Clifford Function Reference

sum

Sum of elements.
(Clifford overloading of standard MATLAB® function)

Description

sum(x) computes the sum of all elements in object x.

Examples

>> x = clifford(1,2,3,4)
 
x =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 + 4.0000 e12
 
>> y = clifford(4,3,2,1)
 
y =
 
   4.0000 e0
 + 3.0000 e1 + 2.0000 e2
 + 1.0000 e12
 
>> z = [x,y]
 
z =
 
     1x2 Cl(1,1) multivector array
 
>> sum(z)
 
ans =
 
   5.0000 e0
 + 5.0000 e1 + 5.0000 e2
 + 5.0000 e12
