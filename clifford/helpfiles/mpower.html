Clifford Function Reference

mpower

^ Matrix power.
(Clifford overloading of standard MATLAB® function)

Description

mpower(x,y) raises matrix x to the power y.

Examples

>> clifford_signature(1,1)
>> y = [3*e1,0;0,2*e12]
 
y =
 
     2x2 Cl(1,1) multivector array
	 
>> show(y)
 
y =
 
e0 *
 
     0     0
     0     0

+ e1 *
 
     3     0
     0     0

+ e12 *
 
     0     0
     0     2
 
>> mpower(y,2)
 
ans =
 
     2x2 Cl(1,1) multivector array
 
>> show(ans)
 
ans =
 
e0 *
 
     9     0
     0     4

>> mpower(y,3)
 
ans =
 
     2x2 Cl(1,1) multivector array
 
>> show(ans)
 
ans =
 
e1 *
 
    27     0
     0     0

+ e12 *
 
     0     0
     0     8
