Clifford Function Reference

grade_four_involution

Computes the grade 4 involution of a clifford multivector. This function negates the grade four elements of a multivector.

Description

grade_four_involution(x) negates the grade four elements of a multivector x.

Examples

In this example the pseudoscalar is the only grade 4 element, hence only the pseudoscalar element is negated. 
>> x = randm
 
x =
 
 - 0.1564 e0  
 - 0.0418 e1   + 0.4945 e2   - 0.2878 e3   + 0.2623 e4  
 + 0.1032 e12  - 0.0782 e13  - 0.3534 e14  - 0.0950 e23  - 0.0290 e24  - 0.4913 e34 
 + 0.0643 e123 - 0.2749 e124 - 0.0315 e134 + 0.1124 e234
 - 0.3025 e1234
 
>> grade_four_involution(x)
 
ans =
 
 - 0.1564 e0  
 - 0.0418 e1   + 0.4945 e2   - 0.2878 e3   + 0.2623 e4  
 + 0.1032 e12  - 0.0782 e13  - 0.3534 e14  - 0.0950 e23  - 0.0290 e24  - 0.4913 e34 
 + 0.0643 e123 - 0.2749 e124 - 0.0315 e134 + 0.1124 e234
 + 0.3025 e1234
