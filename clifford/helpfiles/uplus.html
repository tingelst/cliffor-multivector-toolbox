Clifford Function Reference

uplus

+ Unary plus.
(Clifford overloading of standard MATLAB® function)

Description

uplus(x) is an alternative for using the statement +x.

Examples

>> clifford_signature(1,1)
>> x = clifford(-1,2,-3,4)
 
x =
 
 - 1.0000 e0
 + 2.0000 e1 - 3.0000 e2
 + 4.0000 e12
 
>> uplus(x)
 
ans =
 
 - 1.0000 e0
 + 2.0000 e1 - 3.0000 e2
 + 4.0000 e12
