Clifford Function Reference

lu

Lu decomposition of a matrix of multivectors.
(Clifford overloading of standard MATLAB® function)

Description

[y,z] = lu(x) decomposes the matrix x into a lower triangular matrix y and an upper triangular matrix z.

Examples

>> clifford_signature(1,1)
>> x = randm(3)
 
x =
 
     3x3 Cl(1,1) multivector array
