Clifford Function Reference

grade

Extracts the y-th grade component of a clifford multivector. The result may be empty (example: grade(e1, 2) = []).

Description

grade(x,y) returns the elements of multivector x, which are part of the grade, y.

Examples

>> x = randm
 
x =
 
 - 0.8189 e0
 + 0.5730 e1 + 0.0000 e2
 - 0.0340 e12
 
>> grade(x,0)
 
ans =
 
 - 0.8189 e0
 
>> grade(x,1)
 
ans =
 
   0.5730 e1 + 0.0000 e2
 
>> grade(x,2)
 
ans =
 
 - 0.0340 e12
