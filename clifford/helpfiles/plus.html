Clifford Function Reference

plus

+ Plus.
(Clifford overloading of standard MATLAB® function)

Description

plus(x,y) adds element x to element y.

Examples

>> clifford_signature(2,2)
>> e123 + 3*e4
 
ans =
 
   3.0000 e4  
 + 1.0000 e123
 
>> x = randm
 
x =
 
 - 0.0854 e0
 - 0.3905 e1 - 0.2916 e2
 - 0.8690 e12
 
>> y = randm
 
y =
 
 - 0.4205 e0
 - 0.6912 e1 - 0.2524 e2
 - 0.5308 e12
 
>> plus(x,y)
 
ans =
 
 - 0.5059 e0
 - 1.0817 e1 - 0.5440 e2
 - 1.3998 e12
