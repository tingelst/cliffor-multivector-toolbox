Clifford Function Reference

uminus

- Unary minus.
(Clifford overloading of standard MATLAB® function)

Description

uminus(x) reverses the signs of the input argument, x and is an alternative of doing -x.

Examples

>> clifford_signature(1,1)
>> x = randm
 
x =
 
   0.6375 e0
 + 0.4934 e1 - 0.2405 e2
 + 0.5407 e12
 
>> uminus(x)
 
ans =
 
 - 0.6375 e0
 - 0.4934 e1 + 0.2405 e2
 - 0.5407 e12
