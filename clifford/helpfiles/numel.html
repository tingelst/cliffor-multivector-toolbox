Clifford Function Reference

numel

Number of elements in an array or subscripted array expression.
(Clifford overloading of standard MATLAB® function)

Description

numel(x) returns the number of elements in an array x.

Examples

>> x = randm(2)
 
x =
 
     2x2 Cl(1,1) multivector array
 
>> numel(x)

ans =

     4
