Clifford Function Reference

squeeze

Remove singleton dimensions.
(Clifford overloading of standard MATLAB® function)

Description

squeeze(x) removes all singleton dimensions from array x.

Examples

>> clifford_signature(1,1)
>> x = randm(4,1,2)
 
x =
 
     4x1x2 Cl(1,1) multivector array
 
>> squeeze(x)
 
ans =
 
     4x2 Cl(1,1) multivector array
