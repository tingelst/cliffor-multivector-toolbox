Clifford Function Reference

mean

Average or mean value
(Clifford overloading of standard MATLAB® function)

Syntax

Y = mean(X, dim)

Description

mean(X, dim) computes the mean values of the elements of the multivector array X along dimension dim. If omitted, dim defaults to the columns of X.

Examples

>> A = [e1 e2; e2 e1]
 
A = 2x2 Cl(0,3) multivector array
 
>> show(mean(A))
 
e1 *
 
    0.5000    0.5000

+ e2 *
 
    0.5000    0.5000
