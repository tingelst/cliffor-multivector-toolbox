Clifford Function Reference

eq

== Equal.
(Clifford overloading of standard MATLAB® function)

Description

eq(x,y) returns a logical true or false value based on whether the input values x and y are equal. It can also be used to determine equality of array values.

Examples

>> x = randm
 
x =
 
 - 0.7303 e0
 + 0.6393 e1 + 0.2399 e2
 - 0.0199 e12
 
>> y = randm
 
y =
 
   0.0811 e0
 - 0.6956 e1 - 0.0376 e2
 + 0.7129 e12
 
>> eq(x,x)

ans =

     1

>> eq(x,y)

ans =

     0
Array equality: 
>> a  = [x,y,x,y]
 
a =
 
     1x4 Cl(1,1) multivector array
	 
>> z = [x,y,y,x]
 
z =
 
     1x4 Cl(1,1) multivector array
	 
>> eq(a,z)

ans =

     1     1     0     0
