Clifford Function Reference

subsref

Subscripted reference
(Clifford overloading of standard MATLAB® function)

Description

x = subsref(a, ss) is an overloading of a MATLAB® function that plays an essential role in subscripted references. See the documentation on the MATLAB® function for details, and the source code of the Clifford implementation.
