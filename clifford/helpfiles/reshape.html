Clifford Function Reference

reshape

Change size.
(Clifford overloading of standard MATLAB® function)

Description

reshape(x,y,z) resizes array x into an array of dimensions y by z as long as there is the same number of elements.

Examples

>> x = randm(4)

     4x4 Cl(1,1) multivector array
