Clifford Function Reference

abs

Absolute value, or modulus of a multivector.
(Clifford overloading of standard MATLAB® function)

Description

abs(x) returns the absolute value, or modulus of its argument, x. Applied to a matrix of multivectors it returns matrix of the moduli of all of its entries.

Examples

>> clifford_signature(1,1)
>> x = clifford(1,-2,3,-4)
 
x =
 
   1.0000 e0
 - 2.0000 e1 + 3.0000 e2
 - 4.0000 e12
 
>> abs(x)

ans =

    5.4772

>> y = [x,x;x,x]
 
y =
 
     2x2 Cl(1,1) multivector array
 
>> abs(y)

ans =

    5.4772    5.4772
    5.4772    5.4772
