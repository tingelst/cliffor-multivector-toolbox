Clifford Function Reference

right_contraction

Computes the right contraction of two multivectors.

Description

right_contraction(x,y) computes the right contraction of two multivectors x and y. This is a generalisation of the inner product of vectors. "The right contraction (symbol ⌊) is defined as Ak⌊Bl = ‹AkBl›(k−l)"[1] ,

Examples

>> clifford_signature(3,0)
>> x = e1 + e2
 
x =
 
   1.0000 e1  + 1.0000 e2 
 
>> y = e123
 
y =
 
   1.0000 e123
 
>> right_contraction(x,y)
 
ans =
 
   0.0000 e0 
 
>> right_contraction(y,x)
 
ans =
 
 - 1.0000 e13 + 1.0000 e23
 
>> right_contraction(x,x)
 
ans =
 
   2.0000 e0
