Clifford Function Reference

clifford_signature

Initialises the Clifford Algebra.

Description

clifford_signature(x,y) initalises the signature of the Clifford Algebra being used.

Examples

>> clifford_signature(1,1)
>> clifford_signature(2,2)
>> clifford_signature(7,8)
