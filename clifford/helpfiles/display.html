Clifford Function Reference

display

Display array.
(Clifford overloading of standard MATLAB® function)

Description

display(x) displays the value of the argument, x, formatted in the style of value creation.

Examples

For example if you had previously created value y: 
>> x = randm
 
x =
 
 - 0.9401 e0
 - 0.2775 e1 - 0.0973 e2
 + 0.1723 e12
 
>> display(x)
 
x =
 
 - 0.9401 e0
 - 0.2775 e1 - 0.0973 e2
 + 0.1723 e12
 
>> display(y)
 
y =
 
 - 0.1855 e0
 + 0.3149 e1 + 0.2783 e2
 - 0.8882 e12
 
>> clifford_signature(1,1)
>> x = randm(2)
 
x =
 
     2x2 Cl(1,1) multivector array
 
>> display(x)
 
x =
 
     2x2 Cl(1,1) multivector array
