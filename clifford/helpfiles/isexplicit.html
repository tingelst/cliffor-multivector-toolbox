Clifford Function Reference

isexplicit

True for a clifford multivector without empty components. This means that every component of the multivector contains a numeric array, none is empty.

Description

isexplicit(x) checks that the multivector, x, has no empty components.

Examples

>> x = randm
 
x =
 
   0.2018 e0
 - 0.3179 e1 - 0.5535 e2
 - 0.7429 e12
 
>> isexplicit(x)

ans =

     1

>> y = bivector(x)
 
y =
 
 - 0.7429 e12
 
>> isexplicit(y)

ans =

     0
