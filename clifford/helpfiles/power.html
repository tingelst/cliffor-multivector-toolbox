Clifford Function Reference

power

.^ Array power.
(Clifford overloading of standard MATLAB® function)

Description

power(x,y) raises element x to power y.

Examples

>> clifford_signature(1,1)
>> x = clifford(1,2,3,4)
 
x =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 + 4.0000 e12
 
>> power(x,2)
 
ans =
 
   12.0000 e0
 + 4.0000 e1 + 6.0000 e2
 + 8.0000 e12
 
>> x = randm
 
x =
 
   0.1554 e0
 - 0.4396 e1 + 0.5084 e2
 - 0.7239 e12
 
>> power(x,2)
 
ans =
 
   0.4830 e0
 - 0.1366 e1 + 0.1580 e2
 - 0.2249 e12
