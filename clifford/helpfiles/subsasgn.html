Clifford Function Reference

subsasgn

Subscripted assignment
(Clifford overloading of standard MATLAB® function)

Description

x = subsasgn(a, ss, b) is an overloading of a MATLAB® function that plays an essential role in subscripted assignment. See the documentation on the MATLAB® function for details, and the source code of the Clifford implementation.
