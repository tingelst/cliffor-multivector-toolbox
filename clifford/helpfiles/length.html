Clifford Function Reference

length

Length of vector or length of longest array dimension.
(Clifford overloading of standard MATLAB® function)

Description

length(x) returns the value of the length of the argument, x.

Examples

>> clifford_signature(1,1)
>> x = randm
 
x =
 
 - 0.1164 e0
 - 0.0734 e1 - 0.6395 e2
 - 0.7563 e12
 
>> length(x)

ans =

     1

>> y = randm(2,3)
 
y =
 
     2x3 Cl(1,1) multivector array
 
>> length(y)

ans =

     3
