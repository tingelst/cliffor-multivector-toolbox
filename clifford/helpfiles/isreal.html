Clifford Function Reference

isreal

True for real (clifford) array.
(Clifford overloading of standard MATLAB® function)

Description

isreal(x) checks that all elements of a Clifford array are real values and then returns logical true or false (1 or 0).

Examples

>> clifford_signature(1,1)
>> x = randm
 
x =
 
 - 0.0801 e0
 - 0.8716 e1 - 0.2931 e2
 - 0.3847 e12
 
>> isreal(x)

ans =

     1

>> x = e1 + i.*e2
 
x =
 
   1.0000 e1 + (0.0000+1.0000i) e2
 
>> isreal(x)

ans =

     0
