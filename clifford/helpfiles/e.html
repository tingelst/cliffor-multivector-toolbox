Clifford Function Reference

e0,e1 etc

Clifford basis blades
(Clifford overloading of standard MATLAB® function)

Description

These functions represent the clifford basis blades e0, e1, e2, ..., e123. They are implemeted as parameter less functions which are initialised by clifford_signature. Note that there may be a very large number of these functions if large algebras are initialised.

Examples

>> clifford_signature(3,3)
>> e1234
 
ans =
 
   1.0000 e1234
