Clifford Function Reference

left_contraction

Computes the left contraction of two multivectors.

Description

left_contraction(x,y) computes the left contraction of two multivectors, x and y. This is a generalisation of the inner product of vectors. "In general the left contraction (symbol ⌋) of a k-vector Ak = ‹A›k with a l-vector Bl = ‹B›l is defined as Ak⌋Bl = ‹AkBl›(l−k)"[1]

Examples

>> clifford_signature(3,0)
>> x = e1 + e2
 
x =
 
   1.0000 e1  + 1.0000 e2 
 
>> y = e123
 
y =
 
   1.0000 e123
 
>> left_contraction(x,y)
 
ans =
 
 - 1.0000 e13 + 1.0000 e23
 
>> left_contraction(y,x)
 
ans =
 
   0.0000 e0 
 
>> left_contraction(x,x)
 
ans =
 
   2.0000 e0
