Clifford Multivector Toolbox

For a general overview of Clifford algebra, and how it is handled by the toolbox, see the overview page. For a detailed list of functions, see the alphabetical or categorized lists.

Many Matlab® functions will work with Clifford Algebra, because all the functions called are overloaded by the toolbox, and the semantics of the multivector version are consistent with what the Matlab® function expects. A list of these functions is available.
