Clifford Function Reference

dual

Computes the dual of a multivector.

Description

dual(x) computes the dual of a multivector, x, by division with the unit pseudoscalar.

Examples

>> clifford_signature(3,0)
>> x = randm
 
x =
 
 - 0.0276 e0 
 - 0.6458 e1  - 0.1467 e2  - 0.5996 e3 
 + 0.2808 e12 - 0.2967 e13 + 0.0334 e23
 - 0.1819 e123
 
>> dual(x)
 
ans =
 
 - 0.1819 e0 
 + 0.0334 e1  + 0.2967 e2  + 0.2808 e3 
 + 0.5996 e12 - 0.1467 e13 + 0.6458 e23
 + 0.0276 e123
