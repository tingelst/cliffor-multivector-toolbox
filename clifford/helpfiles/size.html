Clifford Function Reference

size

Size of matrix.
(Clifford overloading of standard MATLAB® function)

Description

size(x) returns the dimensions of a matrix x.

Examples

>> clifford_signature(1,1)
>> x = randm(4)
 
x =
 
     4x4 Cl(1,1) multivector array
 
>> size(x)

ans =

     4     4

>> x = randm(4)
 
x =
 
     4x4 Cl(1,1) multivector array
 
>> [y,z] = size(x)

y =

     4


z =

     4
