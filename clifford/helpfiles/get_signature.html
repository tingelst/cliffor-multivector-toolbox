Clifford Function Reference

get_signature

Returns the signature of a multivector.

Description

get_signature(x) returns the Clifford signature of the multivector argument x.

Examples

>> x = randm
 
x =
 
   0.3540 e0
 + 0.6497 e1 - 0.5681 e2
 - 0.3603 e12
 
>> get_signature(x)

ans =

      1      1      0

>> clifford_signature(2,2)
>> y = randm
 
y =
 
   0.4060 e0  
 - 0.4531 e1   + 0.4473 e2   + 0.0677 e3   + 0.2006 e4  
 - 0.3339 e12  - 0.1184 e13  - 0.0558 e14  + 0.0848 e23  - 0.3351 e24  + 0.0946 e34 
 - 0.2433 e123 + 0.0133 e124 + 0.1309 e134 + 0.0656 e234
 + 0.2172 e1234
 
>> get_signature(y)

ans =

      2      2      0
