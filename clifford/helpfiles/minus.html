Clifford Function Reference

minus

Subtraction.
(Clifford overloading of standard MATLAB® function)

Description

minus(x,y) subtracts multivector y from multivector x.

Examples

>> clifford_signature(1,1)
>> e1 - e2
 
ans =
 
   1.0000 e1 - 1.0000 e2
   
>> minus(e1,e2)
 
ans =
 
   1.0000 e1 - 1.0000 e2
