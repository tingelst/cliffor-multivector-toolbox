Clifford Function Reference

cast

Cast multivector variable to different data type.
(Clifford overloading of standard MATLAB® function)

Description

cast(x,class) casts Clifford object, x, into a different data type, for example changing the data type of the numerical basis coefficients of a multivector.

Examples

>> clifford_signature(0,2)
>> x = randm
 
x =
 
   0.1389 e0
 - 0.3224 e1 + 0.5852 e2
 - 0.7309 e12
 
>> cast(x,'int8')
 
ans =
 
   0.0000 e0
 + 0.0000 e1 + 1.0000 e2
 - 1.0000 e12
