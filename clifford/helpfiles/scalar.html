Clifford Function Reference

scalar

Extracts the scalar component of a clifford multivector. The result is numeric, not a multivector.

Description

scalar(x) extracts the scalar component of a multivector x.

Examples

>> clifford_signature(1,1)
>> x = clifford(1,2,3,4)
 
x =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 + 4.0000 e12
 
>> scalar(x)
 
ans =
 
   1.0000 e0
 
>> y = randm
 
y =
 
 - 0.4631 e0
 + 0.6978 e1 + 0.5373 e2
 - 0.0999 e12
 
>> scalar(y)
 
ans =
 
 - 0.4631 e0
