Clifford Function Reference

ndims

Number of array dimensions.
(Clifford overloading of standard MATLAB® function)

Description

ndims(x) returns the number of dimensions within the argument array x.

Examples

>> clifford_signature(1,1)
>> x = randm(2,2,3)
 
x =
 
     2x2x3 Cl(1,1) multivector array
 
>> ndims(x)

ans =

     3
